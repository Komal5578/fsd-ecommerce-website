# Blushberry Gifts 🎁

A minimalistic e-commerce web application for handcrafted gifts built with React + Vite.

## Features

- **Landing Page** - Beautiful hero section with "Handcrafted Gifts That Never Disappoint" tagline
- **Product Catalog** - Browse 14 handcrafted gift items with images, descriptions, and prices
- **Shopping Cart** - Add/remove items, view cart total, toggle cart sidebar
- **Checkout** - Order confirmation modal on successful checkout


## Tech Stack

- React 18
- Vite
- CSS3 (Custom styling with Ephesis Google Font)
